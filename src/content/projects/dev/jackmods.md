A web-based interface for modifying Jackbox Party Pack games. Only supports Blather 'Round. By uploading the default word list, you can make modifications and explore how the different prompts are implemented. Alternatively, you can also create your own prompt list from scratch and replace the game's default list. The specific file you need (included in the game in a hidden folder) is blankyblankpasswords.jet.
