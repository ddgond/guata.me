# helloquiz setup — landkreise v1

Generated by `scripts/helloquiz-snapshot.mjs`. One row per quiz to create on
[helloquiz.app](https://helloquiz.app); quiz titles and descriptions are yours
to write on the site — the Coverage column is just the label the same drill
has on guata.me.

**GeoJSON URL** (must be deployed before helloquiz can fetch it):

    https://guata.me/data/helloquiz/landkreise-v1.json

This snapshot is frozen: helloquiz references shapes by index, so the file
must never change. Updated data means a new version and repointing.

## Steps

1. Create the base quiz as a map quiz with the GeoJSON URL above, no shape
   filter, and questions auto-imported from the `name` property.
2. Create each band and region quiz as a **remix of the base quiz**, changing
   only the shape filter to the single property=value below.
3. After each import, check the shape count against the table.
